# 2026年分 所得税公式確認レポート

確認日: 2026-06-18  
対象: 2026年分の所得税、給与収入のみ、会社員、単身、扶養なし  
対象外: 住民税、社会保険料、画面表示  
方針: 国税庁または財務省の公式資料だけを使用し、公式確認できない値は変更対象にしない。

## 結論

年収5,000,000円のケースでは、現在のアプリ計算は所得税の基礎控除を480,000円としているが、国税庁資料では2026年分（令和8年分）の該当する基礎控除は680,000円と確認できる。

給与所得控除は、年収5,000,000円では現行アプリの1,440,000円と公式確認値が一致する。所得税の速算表、復興特別所得税2.1%、社会保険料控除の全額控除、課税所得の1,000円未満切捨ては公式確認できる。

所得税額の最終的な円未満端数処理については、今回確認した国税庁ページでは、アプリが採用している「所得税＋復興特別所得税を四捨五入して1円単位にする」処理を直接確認できなかった。そのため、この端数処理は変更しない前提で差額を算出する。

## 年収5,000,000円の計算過程

前提として、今回の対象外である社会保険料は現在のアプリ計算値をそのまま使用する。

- 給与収入: 5,000,000円
- 現在のアプリ計算による社会保険料本人負担: 757,336円

### 1. 給与所得控除

年収5,000,000円は、国税庁 No.1410 の「3,600,001円から6,600,000円まで」に該当する。

```text
給与所得控除 = 給与収入 x 20% + 440,000円
             = 5,000,000円 x 20% + 440,000円
             = 1,000,000円 + 440,000円
             = 1,440,000円
```

### 2. 給与所得

```text
給与所得 = 給与収入 - 給与所得控除
         = 5,000,000円 - 1,440,000円
         = 3,560,000円
```

### 3. 基礎控除

給与収入のみのため、合計所得金額は給与所得3,560,000円とみなす。これは「336万円超489万円以下」に該当するため、令和8年分の基礎控除は680,000円。

```text
基礎控除 = 680,000円
```

### 4. 社会保険料控除

社会保険料控除は、今回対象外の社会保険料計算を変更しないため、現在のアプリ計算値を使用する。

```text
社会保険料控除 = 757,336円
```

### 5. 課税所得

```text
課税所得（切捨て前） = 給与所得 - 基礎控除 - 社会保険料控除
                    = 3,560,000円 - 680,000円 - 757,336円
                    = 2,122,664円

課税所得（1,000円未満切捨て後） = 2,122,000円
```

### 6. 所得税本税

課税所得2,122,000円は、国税庁 No.2260 の速算表で「1,950,000円から3,299,000円まで」に該当する。

```text
所得税本税 = 課税所得 x 10% - 97,500円
           = 2,122,000円 x 10% - 97,500円
           = 212,200円 - 97,500円
           = 114,700円
```

### 7. 復興特別所得税

```text
復興特別所得税 = 所得税本税 x 2.1%
               = 114,700円 x 2.1%
               = 2,408.7円
```

### 8. 所得税と復興特別所得税の合計

```text
合計（円未満処理前） = 所得税本税 + 復興特別所得税
                    = 114,700円 + 2,408.7円
                    = 117,108.7円
```

アプリの現行端数処理（四捨五入して1円単位）を維持する場合:

```text
所得税・復興特別所得税合計 = 117,109円
```

## 現在のアプリ計算との差額

現在のアプリ計算値:

```text
給与所得控除 = 1,440,000円
給与所得 = 3,560,000円
基礎控除 = 480,000円
社会保険料控除 = 757,336円
課税所得 = 2,322,000円
所得税・復興特別所得税合計 = 137,529円
```

公式確認値を反映し、アプリの現行端数処理を維持した場合:

```text
給与所得控除 = 1,440,000円
給与所得 = 3,560,000円
基礎控除 = 680,000円
社会保険料控除 = 757,336円
課税所得 = 2,122,000円
所得税・復興特別所得税合計 = 117,109円
```

差額:

```text
所得税差額 = 117,109円 - 137,529円
           = -20,420円
```

つまり、年収5,000,000円では、2026年分の基礎控除680,000円を反映すると、現在のアプリ計算より所得税・復興特別所得税が20,420円少なくなる。

## コード変更判断

今回は検証レポートのみ作成し、コード・住民税・社会保険料・画面表示は変更しない。

次にコードへ反映する場合、公式確認済みとして変更候補になるのは所得税の基礎控除テーブルである。給与所得控除は年収5,000,000円では現行計算と一致するが、低年収帯の最低保障額は現行設定と公式確認値が異なるため、所得税計算全体へ反映する際には別途テーブル更新が必要になる。
