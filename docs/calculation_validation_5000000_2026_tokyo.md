## 項目別検証一覧

### 1. 所得税

| 観点 | 内容 |
| --- | --- |
| 現行アプリの結果 | 137,222円 |
| 使用している率・控除額 | 給与所得控除: 5,000,000円 x 20% + 440,000円、所得税基礎控除: 480,000円、所得税率: 10%、速算控除: 97,500円、復興特別所得税率: 2.1% |
| 計算式 | 給与所得控除 = 5,000,000円 x 20% + 440,000円 = 1,440,000円 |
|  | 給与所得 = 5,000,000円 - 1,440,000円 = 3,560,000円 |
|  | 課税所得 = 3,560,000円 - 480,000円 - 760,592円 = 2,319,408円 |
|  | 1,000円未満切り捨て後 = 2,319,000円 |
|  | 所得税本税 = 2,319,000円 x 10% - 97,500円 = 134,400円 |
|  | 復興特別所得税込み = 134,400円 x 1.021 = 137,222.4円 |
|  | 現行アプリ結果 = round(137,222.4円) = 137,222円 |
| 公式出典 | 国税庁 No.1410 給与所得控除: https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1410.htm |
|  | 国税庁 No.1199 基礎控除: https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1199.htm |
|  | 国税庁 No.2260 所得税の税率: https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2260.htm |
| 仮値か確定値か | 現行アプリ上は仮値。税率・復興特別所得税率は公式表と照合可能。基礎控除480,000円は2026年分として差し替え候補。 |
