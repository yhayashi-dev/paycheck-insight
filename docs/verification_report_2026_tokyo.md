## 年収5,000,000円ケースの計算過程

### 前提

- 年収: 5,000,000円
- 月額給与: 5,000,000円 ÷ 12 = 416,666.666...円
- アプリ表示用月額給与: round(416,666.666...) = 416,667円
- 標準報酬月額判定用月額: int(416,666.666...) = 416,666円
- 年齢: 52歳のため介護保険あり

### 給与所得

年収5,000,000円は、現在設定の給与所得控除テーブルでは「3,600,001円から6,600,000円まで」に該当します。

```text
給与所得控除 = 5,000,000円 x 20% + 440,000円
             = 1,000,000円 + 440,000円
             = 1,440,000円

給与所得 = 5,000,000円 - 1,440,000円
         = 3,560,000円
```

### 健康保険

月額416,666円は、健康保険の標準報酬月額表で「395,000円以上425,000円未満」に該当します。

```text
健康保険 標準報酬月額 = 410,000円
健康保険料率 = 9.91%

月額保険料総額 = 410,000円 x 9.91%
              = 40,631円

本人月額 = round(40,631円 ÷ 2)
         = 20,316円

本人年額 = 20,316円 x 12
         = 243,792円

会社年額 = 20,316円 x 12
         = 243,792円
```

検証対象:

- 2026年度の東京都健康保険料率が9.91%でよいか
- 標準報酬月額410,000円の等級判定が正しいか
- 労使折半後の端数処理が `round()` でよいか

### 介護保険

52歳のため、介護保険第2号被保険者として計算しています。

```text
介護保険 標準報酬月額 = 410,000円
介護保険料率 = 1.59%

月額保険料総額 = 410,000円 x 1.59%
              = 6,519円

本人月額 = round(6,519円 ÷ 2)
         = 3,260円

本人年額 = 3,260円 x 12
         = 39,120円

会社年額 = 3,260円 x 12
         = 39,120円
```

検証対象:

- 2026年度の介護保険料率が1.59%でよいか
- 52歳を介護保険対象にする判定は正しいか
- 労使折半後の端数処理が `round()` でよいか

### 厚生年金

月額416,666円は、厚生年金の標準報酬月額表で「395,000円以上425,000円未満」に該当します。

```text
厚生年金 標準報酬月額 = 410,000円
厚生年金保険料率 = 18.3%

月額保険料総額 = 410,000円 x 18.3%
              = 75,030円

本人月額 = round(75,030円 ÷ 2)
         = 37,515円

本人年額 = 37,515円 x 12
         = 450,180円

会社年額 = 37,515円 x 12
         = 450,180円
```

検証対象:

- 厚生年金保険料率18.3%でよいか
- 標準報酬月額410,000円の等級判定が正しいか
- 標準報酬月額の上限650,000円でよいか
- 労使折半後の端数処理が `round()` でよいか

### 雇用保険

```text
雇用保険 労働者負担率 = 0.55%
雇用保険 事業主負担率 = 0.90%

本人年額 = round(5,000,000円 x 0.55%)
         = round(27,500円)
         = 27,500円

会社年額 = round(5,000,000円 x 0.90%)
         = round(45,000円)
         = 45,000円
```

検証対象:

- 2026年度の一般の事業の料率。厚生労働省の令和8年度PDFでは、一般の事業は労働者負担5/1,000、事業主負担8.5/1,000と示されているため、現在設定の0.55%・0.90%は差し替え候補です。
- 雇用保険料の端数処理。

### 社会保険料合計

```text
本人負担合計 = 健康保険 + 介護保険 + 厚生年金 + 雇用保険
            = 243,792円 + 39,120円 + 450,180円 + 27,500円
            = 760,592円
```

### 所得税

```text
給与所得 = 3,560,000円
所得税 基礎控除 = 480,000円
社会保険料控除 = 760,592円

課税所得（丸め前） = 3,560,000円 - 480,000円 - 760,592円
                  = 2,319,408円

課税所得（1,000円未満切り捨て） = 2,319,000円
```

2,319,000円は現在設定の所得税速算表で「1,950,000円から3,299,999円」に該当します。

```text
所得税本税 = 2,319,000円 x 10% - 97,500円
          = 231,900円 - 97,500円
          = 134,400円

復興特別所得税込み = 134,400円 x 1.021
                  = 137,222.4円

所得税 = round(137,222.4円)
       = 137,222円
```

検証対象:

- 2026年分の所得税基礎控除。国税庁 No.1199 では令和8年分の基礎控除が所得金額別に示されており、現在設定の480,000円とは異なる可能性があります。
- 給与所得控除テーブル。国税庁 No.1410 では令和7年分以降の給与所得控除が示されており、現在設定とは一部異なる可能性があります。
- 復興特別所得税の端数処理。

### 住民税

```text
給与所得 = 3,560,000円
住民税 基礎控除 = 430,000円
社会保険料控除 = 760,592円

課税所得（丸め前） = 3,560,000円 - 430,000円 - 760,592円
                  = 2,369,408円

課税所得（1,000円未満切り捨て） = 2,369,000円

所得割 = 2,369,000円 x 10%
       = 236,900円

均等割 = 5,000円
森林環境税 = 1,000円

住民税 = 236,900円 + 5,000円 + 1,000円
       = 242,900円
```

検証対象:

- 2026年度の住民税基礎控除、所得割、均等割、森林環境税。
- 調整控除を入れるかどうか。
- 非課税判定を入れるかどうか。
- 住民税は前年所得に対する課税であるため、本アプリで「入力年収を前年所得相当として扱う」前提を明記するか。

### 会社負担分

```text
健康保険 会社負担 = 243,792円
介護保険 会社負担 = 39,120円
厚生年金 会社負担 = 450,180円
雇用保険 会社負担 = 45,000円

子ども・子育て拠出金 = round(410,000円 x 0.36% x 12)
                    = round(17,712円)
                    = 17,712円

会社負担分 = 243,792円 + 39,120円 + 450,180円 + 45,000円 + 17,712円
          = 795,804円
```

検証対象:

- 会社負担分に子ども・子育て拠出金を含める表示方針。
- 2026年度の子ども・子育て拠出金率。
- 各会社負担項目の端数処理。

### 手取り・総人件費

```text
税金合計 = 所得税 + 住民税
        = 137,222円 + 242,900円
        = 380,122円

年間手取り = 年収 - 社会保険料本人負担 - 税金合計
          = 5,000,000円 - 760,592円 - 380,122円
          = 3,859,286円

月平均手取り = round(3,859,286円 ÷ 12)
            = 321,607円

総人件費 = 年収 + 会社負担分
        = 5,000,000円 + 795,804円
        = 5,795,804円
```
