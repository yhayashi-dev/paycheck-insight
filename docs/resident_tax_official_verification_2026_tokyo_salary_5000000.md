# 2026年度 住民税公式確認レポート

確認日: 2026-06-18  
対象: 東京都、52歳、単身、扶養なし、会社員、給与収入のみ、年収5,000,000円、賞与なし  
前提: 2026年度住民税は前年所得課税のため、前年（2025年、令和7年）も同じ年収5,000,000円だったものとして試算する。  
対象外: 所得税、社会保険料、画面表示  
方針: 東京都・自治体・総務省・国税庁・e-Gov法令検索などの公式資料だけを使用し、まず検証レポートのみ作成する。

## 結論

現在のアプリの住民税は243,200円。公式確認値で、現行アプリの社会保険料本人負担757,336円をそのまま使うと、2026年度の住民税は239,700円となる。

差額は次のとおり。

```text
公式確認値 239,700円 - 現在のアプリ 243,200円 = -3,500円
```

差額の主因:

- 現在のアプリは調整控除2,500円を反映していない。
- 現在のアプリは均等割を5,000円としているが、東京都公式資料では令和6年度以後の個人住民税均等割は4,000円（都民税1,000円、区市町村民税3,000円）。森林環境税1,000円は均等割とは別に併せて課税される。

## 東京23区と東京都全体の違い

所得割の標準的な税率は、東京都全体で都民税4%、区市町村民税6%、合計10%。東京23区では区市町村民税に相当する部分が特別区民税として扱われるが、今回の給与収入のみの試算では、所得割の合計税率10%、均等割合計4,000円、森林環境税1,000円は同じ扱いになる。

差が出る可能性があるのは非課税判定。東京都主税局は、東京23区内の場合の均等割非課税基準を示したうえで、23区外では均等割額が非課税となる合計所得金額が異なる場合があるため居住市町村へ確認するよう案内している。今回の年収5,000,000円・給与所得3,560,000円では、23区の非課税基準45万円を大きく超えるため、非課税には該当しない。

## 年収5,000,000円の計算過程

今回の社会保険料計算は対象外のため、現在のアプリ計算値をそのまま使う。

```text
給与収入 = 5,000,000円
社会保険料本人負担 = 757,336円
```

### 1. 給与所得控除

年収5,000,000円は、国税庁 No.1410 の「3,600,001円から6,600,000円まで」に該当する。

```text
給与所得控除 = 給与収入 x 20% + 440,000円
             = 5,000,000円 x 20% + 440,000円
             = 1,000,000円 + 440,000円
             = 1,440,000円
```

### 2. 給与所得

```text
給与所得 = 給与収入 - 給与所得控除
         = 5,000,000円 - 1,440,000円
         = 3,560,000円
```

### 3. 非課税判定

東京23区、単身、扶養なしの場合:

```text
所得割・均等割とも非課税となる基準 = 合計所得金額45万円以下
所得割のみ非課税となる基準 = 総所得金額等45万円以下
今回の合計所得金額 = 3,560,000円
```

3,560,000円は45万円を超えるため、非課税には該当しない。

### 4. 所得控除

```text
基礎控除 = 430,000円
社会保険料控除 = 757,336円
```

今回の対象条件では、扶養控除・配偶者控除などは0円。

### 5. 課税標準

```text
課税標準（切捨て前） = 給与所得 - 基礎控除 - 社会保険料控除
                    = 3,560,000円 - 430,000円 - 757,336円
                    = 2,372,664円

課税標準（1,000円未満切捨て後） = 2,372,000円
```

### 6. 所得割

```text
所得割（調整控除前） = 課税標準 x 10%
                    = 2,372,000円 x 10%
                    = 237,200円

内訳:
都民税分 = 2,372,000円 x 4% = 94,880円
区市町村民税分 = 2,372,000円 x 6% = 142,320円
```

### 7. 調整控除

今回の人的控除額の差は、少なくとも基礎控除差額を使う。令和7年分所得税の基礎控除は、給与所得3,560,000円の場合680,000円。住民税の基礎控除は430,000円。

```text
人的控除額の差 = 680,000円 - 430,000円
              = 250,000円
```

合計課税所得金額が200万円超のため、東京都主税局の式により次のように計算する。

```text
調整控除の原則計算 = {人的控除額の差 - (合計課税所得金額 - 2,000,000円)} x 5%
                  = {250,000円 - (2,372,000円 - 2,000,000円)} x 5%
                  = (250,000円 - 372,000円) x 5%
                  = マイナス
```

2,500円未満となるため、最低額2,500円を適用する。

```text
調整控除 = 2,500円
内訳:
都民税分 = 1,000円
区市町村民税分 = 1,500円
```

```text
所得割（調整控除後） = 237,200円 - 2,500円
                    = 234,700円
```

### 8. 均等割と森林環境税

```text
均等割 = 4,000円
      = 個人都民税1,000円 + 個人区市町村民税3,000円

森林環境税 = 1,000円
```

### 9. 住民税・森林環境税合計

```text
住民税・森林環境税合計 = 所得割（調整控除後） + 均等割 + 森林環境税
                      = 234,700円 + 4,000円 + 1,000円
                      = 239,700円
```

100円未満の端数はないため、税額端数処理後も239,700円。

```text

```

公式確認値:

```text
給与所得 = 3,560,000円
住民税基礎控除 = 430,000円
社会保険料控除 = 757,336円
課税標準 = 2,372,000円
```

```text
239,700円 - 243,200円 = -3,500円
```

内訳:


## コード変更判断

今回は検証レポートのみ作成し、コード・所得税・社会保険料・画面表示は変更しない。

次にコードへ反映する場合、住民税について変更候補となるのは以下。

- 均等割を5,000円から4,000円へ変更
- 森林環境税1,000円は別項目として維持
- 調整控除2,500円の反映
- 税額の100円未満切捨て
- 23区外を扱う場合は、非課税判定を自治体別に持てる構造にする
