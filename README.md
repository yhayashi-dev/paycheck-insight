# paycheck-insight

2026年の会社員向けに、税金・社会保険料・手取り額を概算する Streamlit アプリです。

## 公開アプリ

https://paycheck-insight-dfnvaodwxttypa7gtdouhj.streamlit.app

現在対応している都道府県は東京都・大阪府です。対象条件は、52歳・単身・扶養なし・会社員・協会けんぽ・給与収入のみ・賞与なし・年収を12か月均等支給に固定しています。

## 概要

年収を入力すると、以下を試算します。

画面で東京都または大阪府を選択できます。大阪府の住民税は大阪市を居住地として概算します。

- 所得税
- 住民税
- 健康保険
- 介護保険
- 厚生年金
- 雇用保険
- 社会保険料の本人負担合計
- 会社負担分
- 年間手取り
- 月平均手取り
- 総人件費

また、年収200万円から1000万円まで100万円刻みの一覧も表示します。

## 使い方

依存関係をインストールします。

```bash
pip install -r requirements.txt
```

Streamlit アプリを起動します。

```bash
streamlit run app.py
```

テストを実行します。

```bash
PYTHONPATH=. pytest
```

## プロジェクト構成

```text
paycheck-insight/
  app.py
  requirements.txt
  data/
    rates_2026_osaka.json
    rates_2026_tokyo.json
  src/
    calculators/
    services/
    ui/
  tests/
  docs/
```

主な役割は以下です。

- `app.py`: Streamlit 画面
- `data/rates_2026_tokyo.json`: 東京都の料率、控除額、地域依存項目、確認状況、出典情報
- `data/rates_2026_osaka.json`: 大阪府・大阪市の地域差分、確認状況、出典情報
- `src/calculators/`: 給与所得、所得税、住民税、社会保険料の計算
- `src/services/`: 料率読み込みとシミュレーション
- `src/ui/`: 表示用テーブル整形
- `tests/`: 計算ロジックとアプリ統合テスト
- `docs/`: 公式確認レポート

## 確認状況

アプリ画面上部に「一部未確認項目あり」と表示し、折りたたみの「確認状況と出典」で確認済み項目、未確認項目、適用年度、出典URLを確認できます。

現在の設定フラグは以下です。

| 区分 | provisional |
| --- | --- |
| 給与所得控除 | `false` |
| 所得税 | `true` |
| 住民税 | `false` |
| 社会保険料 | `true` |

所得税は多くの項目を公式確認済みですが、所得税額の最終端数処理が未確認のため、区分全体の `provisional` は `true` のままにしています。社会保険料率の対象6項目は公式確認済みです。ただし、別制度の子ども・子育て支援金0.23%は現在の計算に未反映のため、社会保険料区分は `true` を維持しています。

## 残っている未確認事項

- 所得税額の最終端数処理
- 子ども・子育て支援金率0.23%の計算反映

## 注意事項

- 本アプリは概算ツールです。実際の税額、保険料、手取り額を保証するものではありません。
- 現在対応している都道府県は東京都・大阪府です。大阪府の住民税は大阪市を前提とします。
- 年齢・世帯条件は52歳・単身・扶養なし・協会けんぽ・給与収入のみ・賞与なしに固定しています。
- 健康保険料率と住民税設定は地域依存項目として管理しています。
- 住民税は前年所得課税のため、入力年収を前年も同じだったものとして概算しています。
- 年末調整、追加控除、扶養控除、配偶者控除、所得金額調整控除、賞与、自治体差、勤務先固有の扱いは対象外です。
- 社会保険料は標準報酬月額を年収の12か月均等支給から概算しています。
